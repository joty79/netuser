# Changelog

Όλες οι σημαντικές αλλαγές σε αυτό το project θα καταγράφονται σε αυτό το αρχείο.

Η μορφή βασίζεται στο [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.3.0] - 2026-06-19

### Added
- Οργάνωση και φιλτράρισμα του ιστορικού συνδέσεων ανά δίκτυο (Network ID) χρησιμοποιώντας το `Get-CurrentNetworkIdentity`.
- Εμφάνιση του ονόματος υπολογιστή μαζί με την IP διεύθυνση (`HostName (IP) - user: userName`) στις εγγραφές ιστορικού του TUI μενού.
- Αυτόματη μετατροπή και υποστήριξη παλαιότερων εγγραφών ιστορικού για αποφυγή σφαλμάτων κατά την ανάγνωση.

### Fixed
- Διόρθωση ορθογραφικού σφάλματος διπλού δολαρίου (`$$`) που προκαλούσε την εκτύπωση της συμβολοσειράς `_C.Reset)` κυριολεκτικά στα κάθετα περιγράμματα του TUI scrollable viewer.

## [1.2.0] - 2026-06-19

### Added
- Προσθήκη ιστορικού συνδέσεων (Connection History) το οποίο αποθηκεύεται τοπικά στο `history.json` και εμφανίζεται στο κεντρικό μενού για γρήγορη επανασύνδεση.
- Προσθήκη δυνατότητας εξαγωγής δεδομένων σε CSV και Markdown (.md) πατώντας το πλήκτρο `E` κατά την προβολή των χρηστών. Τα αρχεία αποθηκεύονται στον φάκελο `exports/`.
- Δημιουργία περιγράμματος (boxed border layout) γύρω από το scrollable text viewer για premium εμφάνιση.

### Fixed
- Διόρθωση του πλήκτρου `Esc` που δεν έκλεινε το scrollable viewer (εξαιτίας χρήσης `break` που τερμάτιζε μόνο το `switch` αντί για τη λούπα).
- Διόρθωση του `resize stretch` bug με την εφαρμογή αυστηρού truncation πλάτους γραμμής (πριν την εισαγωγή των ANSI χρωμάτων) ώστε να μην προκαλείται wrapping από το terminal.

## [1.1.0] - 2026-06-19

### Added
- Προσθήκη πλήρους διαδραστικού TUI (Alternate Screen Buffer) χρησιμοποιώντας το `PS_UI_Blueprint.psm1`.
- Προσθήκη παράλληλης σάρωσης δικτύου (Network Discovery) στη θύρα 5985 (WinRM) πατώντας `Ctrl+L` ή μέσω του TUI μενού.
- Υλοποίηση προβολής αποτελεσμάτων σε scrollable text viewer με υποστήριξη πλοήγησης (Up/Down/PageUp/PageDown/Home/End).
- Υποστήριξη αυτόματης εναλλαγής μεταξύ TUI (όταν εκτελείται χωρίς ορίσματα) και CLI (όταν εκτελείται με ορίσματα).

## [1.0.0] - 2026-06-19

### Added
- Δημιουργία του project `netuser`.
- Ανάπτυξη του `Get-NetUsers.ps1` για έλεγχο χρηστών, Administrators, Remote Desktop Users και ενεργών συνεδριών.
- Υποστήριξη τοπικών και απομακρυσμένων queries μέσω WinRM.
- Αυτόματη ρύθμιση του `TrustedHosts` με χρήση `gsudo` για elevation όταν χρειάζεται.
- Δημιουργία `PROJECT_RULES.md` για τη διατήρηση της μνήμης του project.
- Δημιουργία `README.md` για την τεκμηρίωση του project.
